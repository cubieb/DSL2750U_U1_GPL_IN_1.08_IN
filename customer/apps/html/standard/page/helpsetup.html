Wizard

If you are new to networking and this is the first time you configure a router, click "Setup Wizard" and the router will guide you through a few steps to get your network up and running.

Internet Setup

If you are an advanced user and have configured a router before, click "Add" to add an Internet Setup and manually configure all the settings.

Wireless Setting

In the "Wireless" page, you can configure the wireless settings of the router. Note that changes made in this page must also be duplicated on wireless clients that you wish to connect to the wireless network.

To protect your privacy, properly configure the wireless security features.

Wireless Basics
Basic settings to get your wireless network up and running.
Enable Wireless
You can enable or disable the wireless connection feature of the router. If this option is selected, the following parameters are available.
Wireless Network Name (SSID)
When you are searching for available wireless networks, this is the name that appears in the list (unless the "Access Point" is set to "Hidden"). For security purposes, it is recommended to change the pre-configured network name.
Visibility Status
It helps to hide your wireless network. If this option is set to Visible, your wireless network name is broadcast to anyone within the range of your signal. If you are not using encryption, any device that receives your signal can connect to your network. If it is set to Invisible, you must manually enter the Wireless Network Name (SSID) on the client in order to connect to the network.
Country
It identifies the available wireless channel for the country where you are in. In case your country is not in the list, please select the country nearest to you.
Wireless Channel
802.11b and 802.11g use channels to limit interference from other devices. If you are experiencing interference with another 2.4 GHz device such as a baby monitor, security alarm, or cordless phone, try changing the channel on your router.
802.11 Mode
Select "Mixed 802.11g and 802.11b" to operate in b/g mode, or select a specific mode.

Wireless Security
Settings to secure your wireless network
Security Mode
If none of the encryption modes is selected, wireless transmissions to and from your wireless network can be easily intercepted and interpreted by unauthorized users.
WEP

It is a method of encrypting data for wireless communication intended to provide the same level of privacy as a wired network. WEP is not as secure as WPA encryption. To gain access to a WEP network, you must enter the key, which is a string of characters in HEX (hexadecimal - using characters 0-9, A-F) or ASCII (American Standard Code for Information Interchange - alphanumeric characters) format that you create. An ASCII string, which is easier to remember, is converted to HEX for use over the network. When using WEP, you must specify the type of encryption, which determines the length of the key. 128-bit encryption requires a longer key than 64-bit encryption. You can set up to four keys.

Example:
A 64-bit hexadecimal key must contain 10 characters. (12345678FA is a valid string of 10 characters for 64-bit encryption.)
A 128-bit hexadecimal key must contain 26 characters. (456FBCDF123400122225271730 is a valid string of 26 characters for 128-bit encryption.)
A 64-bit ASCII key must contain 5 characters(DMODE is a valid string of 5 characters for 64-bit encryption.)
A 128-bit ASCII key must contain 13 characters (2002HALOSWIN1 is a valid string of 13 characters for 128-bit encryption.)
Auto (WPA or WPA2)
WPA stands for Wi-Fi Protected Access. Auto (WPA or WPA2) uses the TKIP or AES encryption method. It is useful during the transition period of upgrade in the enterprise environment, because it enables "upgraded" users and users not yet "upgraded" to access the network through the router.
WPA Only
It uses Temporal Key Integrity Protocol (TKIP) encryption, instead of WEP, which is disabled.
WPA2 Only
Uses AES (Advanced Encryption Standard) instead of TKIP as its encryption method.
WPA Mode
WPA-PSK
It stands for Wi-Fi Protected Access - Pre-Shared Key. It is WPA for home and SOHO environments, using AES or TKIP encryption, per-packet key construction, and key management that WPA provides in the enterprise environment. The group re-key interval time is required.
WPA-Enterprise
It is Wi-Fi Protected Access - Enterprise. It usually applies to larger Enterprise environments. The information of the RADIUS server and the group re-key interval time must be entered.

Group Key Update Interval: The amount of time before the group key used for broadcast and multicast data is changed.

Authentication Timeout: Authentication Timeout: The amount of time before a client is asked to re-authenticate.

RADIUS Server IP Address: The IP address of the authentication server.

RADIUS Server Port: The port used to connect the authentication server.

RADIUS Server Shared Secret: A passphrase used to verify at the authentication server.

Local Network

LAN Setup
In this page, you can configure the LAN (Local Area Network) interface of the router, including setting the IP Address and subnet mask. The IP address can also used to access the Web management interface. You are recommended to use the default settings if you do not have an existing network.
Router IP Address
The IP address of the router on the local area network. For example, 192.168.0.1. The LAN settings are based on this address.
Subnet Mask
The subnet mask of the router on the local area network.
DHCP Server Settings
DHCP stands for Dynamic Host Configuration Protocol. In this page, you can configure the built-in DHCP server, which assigns IP addresses to computers and other devices on the LAN.
Enable DHCP Server

If the router is properly configured and this option is selected, the DHCP server will manage the IP addresses and other network configuration information of computers and other devices connected to the LAN. Manual configuration is not needed.

The computers (and other devices) connected to your LAN also need to have their TCP/IP configuration set to "DHCP" or "Obtain an IP address automatically".

If Enable DHCP Serveris selected, the following options become available.

DHCP IP Address Range
The two IP addresses (from and to)define the range of IP addresses that the DHCP server uses when assigning addresses to computers and devices on your LAN. Any address that is beyond this range is not managed by the DHCP server, and can therefore, be used by a device whose IP address is manually configured or a device that fails to obtain network address automatically through DHCP.

The router, by default, has the static IP address of 192.168.1.1. This means that addresses 192.168.1.2 to 192.168.1.254 can be available for allocation by the DHCP server.

Example:
Assume that the IP address of the router is 192.168.1.1. A computer, which serves as a Web server, is assigned with the static IP address of 192.168.1.3. Another computer, which serves as an FTP server, is assigned with the static IP address of 192.168.1.4. Therefore, the starting IP address for DHCP IP address range must be 192.168.1.5 or greater.
Example:
Assume that you configure the DHCP Server to manage addresses from 192.168.1.100 to 192.168.1.199. This means that 192.168.1.3 to 192.168.1.99 and 192.168.1.200 to 192.168.1.254 are NOT managed by the DHCP server. Computers or devices that use addresses in these ranges are manually configured.
DHCP Lease Time
The amount of time that a computer can have an IP address before it is required to renew the lease. In the initial lease, the amount of time before the lease expires is specified. If the tenant wishes to retain the address when the lease expires, a new lease can be established. If the lease expires and the address is no longer needed, another tenant can use the address.

Time and Date

With the time configuration function, you can configure, update, and maintain the correct time of the internal system clock. In this page, you can set the time zone that you are in and set the time server.
Current Router Time
It displays the current time on the router. If it is not correct, do as follows to configure the correct time:
Automatically synchronize with Internet Time servers
Select this option if you want to synchronize the clock on the router with a network time server over the Internet. If you are using schedules or logs, this can ensure that the schedules and logs are kept accurately.
NTP Time Server
Select a network time server for synchronization. You can enter the address of a time server or select one from the list. If you have trouble using a server, please select another.
Time Zone
Select your local time zone from the drop-down menu.
